# charitize

Project Description

Charitize is a web-based charity platform that aims to help poor and underprivileged people by making donations simple and accessible.

The website provides an easy way for users to contribute and support social causes. It focuses on creating awareness and encouraging people to help those in need.

🎯 Objectives
To provide a platform for donations
To support underprivileged communities
To promote social responsibility
To create a simple and user-friendly interface
🚀 Features
🏠 Clean and responsive homepage
❤️ Donation section for easy contributions
📖 About section explaining the mission
📞 Contact section for communication
📱 Fully responsive design (mobile + desktop)
💻 Technologies Used
HTML
CSS
JavaScript
